# face-detection-vue

## Tool Usage
  - Face detection is build by **face-api.js**
  - Frontend Framework is build by **Vue.js**

  > For more information, Please find [face-api.js](https://github.com/justadudewhohacks/face-api.js/) & [Vue3](https://v3.vuejs.org/)
